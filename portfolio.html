NAGLREP.com

Custom Wordpress Development

NAGLREP.com is a membership site for LGBT and LGBT-friendly realtors.

Features: Custom payment integration, custom search and review system.

Read More | Visit Site
Payment Integration with FirstData API:
  • Upon successful payment, create a new account and send payment receipt to user and site admin.
  • Wrote cron job to charge annual membership fee automatically when membership expires.
Custom Search for Members
  • Users can search for realtors based on the cities listed on the members' profiles.
  • Search autocomplete populated with the city/state combination in the database.
  • Each city is verified with Google Maps API or in the list of accepted cities created by the site admin.
  • Custom fields in Edit User page for site admin to edit the list of cities for members.
Review System:
  • When new review is submitted, site admin receives an email to approve.
  • Upon approval, the member will receive an email notification linking to the new review.
  • Each member has an average rating associated with their profile.
